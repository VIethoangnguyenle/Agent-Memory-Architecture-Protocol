# Templates — User-facing Templates

Templates để tạo các loại task document. Bản gốc nằm trong `.knowledge-layer/templates/`.

---

## Danh sách Template

| File | Mục đích | Dùng khi |
|------|----------|----------|
| `feature.md` | Checklist feature mới | Task thêm tính năng |
| `fixbug.md` | Template báo cáo & fix bug | Task sửa lỗi |
| `changerequest.md` | Change request | Thay đổi hành vi hiện có |
| `refactor.md` | Refactor task | Tái cấu trúc code |
| `ideation.md` | Ý tưởng thô | Ghi chép ý tưởng ban đầu |

## Template đặc biệt (nằm trong `.knowledge-layer/templates/`)

| File | Mục đích |
|------|----------|
| `knowledge-snapshot.md` | Kiến trúc tổng thể (tích luỹ) |
| `REQUIREMENT.tpl.md` | Template REQUIREMENT.md cho task mới |
| `EXPLORE_CONTEXT.tpl.md` | Template EXPLORE_CONTEXT.md cho task mới |
| `AGENT_TRANSPARENCY.tpl.md` | Template observability log |
